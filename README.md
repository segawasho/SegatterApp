# README

## アプリ説明
ユーザ登録制のコミュニケーションアプリケーションです。
ユーザは投稿、いいね、コメントなどでコミュニケーションを図ることができます。

##URL
https://segatter.herokuapp.com/

## 機能
-新規登録
-ログイン、ログアウト
-投稿
-引用リツイート
-いいね！
-コメント
-プロフィール編集(名前、ID、パスワード、プロフ画像)
